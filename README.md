# Organ-on-a-Chip Monitoring System

## Overview
A full-stack application developed to monitor simulated organ-on-a-chip parameters such as temperature, oxygen level, and glucose concentration. The system uses a Spring Boot backend with a MySQL database and a web-based frontend for data visualization.

## Technologies
- Java, Spring Boot  
- MySQL  
- HTML, CSS, JavaScript  
- GitHub  

## Features
- Simulated sensor data generation  
- REST API-based backend  
- MySQL database integration  
- Web-based data display  

## How to Run
1. Clone the repository  
   git clone https://github.com/jeyalakshmial05-beep/organ-on-a-chip-monitoring-system.git  
2. Configure MySQL in `application.properties`  
3. Run the Spring Boot application  
4. Open frontend files in a browser
